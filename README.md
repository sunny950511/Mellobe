# MELLOBE 제품 홍보 랜딩 페이지

> "지구를 생각하는 마음과 키치한 감성을 담은 친환경 조명 브랜딩 포트폴리오"

## 1. 프로젝트 소개
* **설명:** 버려진 폐페트병을 재활용한 100% 리사이클 자수 원사 갓과 친환경 실리콘 바디를 사용한 장스탠드 조명 브랜드 'MELLOBE'를 홍보하기 위한 인터랙티브 브랜딩 랜딩 페이지입니다.
* **진행 기간:** 2026.05.19 ~ 2026.05.21
* **개발 인원:** 개인 프로젝트 (기여도 100%)

## 2. 배포 및 관련 링크

* **GitHub Repository:** https://github.com/sunny950511-ux/Mellobe

## 3. 사용 기술 스택 

### Frontend
* **React.js & Vite:** React 19 기반의 컴포넌트 아키텍처 및 Vite 8을 통한 초고속 HMR(Hot Module Replacement) 개발 환경 구축.
* **CSS Variables & Keyframe Animation:** 전체 테마(Light/Dark) 시스템 통합 제어 및 젤리 팝핑 인터렉션 등의 키치하고 자연스러운 모션 구현.
* **Intersection Observer API:** 스크롤 높이를 추적해 OOTD 갤러리 영역 진입 시 웹사이트 테마를 자동으로 다크 모드로 반전시키는 스크롤 트리거 기술 적용.
* **Vanilla JS Mouse Trail (Confetti):** 마우스 좌표를 실시간으로 추적하여 다채로운 파스텔 톤의 도형들이 튀어나와 흩어지는 인터랙션 구현.

### 🤖 AI-Assisted Development (Vibe Coding)
* **Google Gemini / Antigravity:** 개발 과정 전반에서 AI와 실시간으로 페어 프로그래밍을 진행하여 물리적 모션 수식 설계 및 복잡한 레이아웃 구조화 단축.
* **AI 활용 목표:** 3D 느낌의 실시간 조명 기울기 및 갓 패턴 시뮬레이터 구현 시 마우스 드래그/입력 좌표 계산의 수식 오류를 줄이고, 단일 파일 내 복잡한 코드를 역할별 컴포넌트로 깔끔하게 리팩토링하는 것을 목표로 설정.

## 4. 사용자 작업 흐름 

**주요 과업: 실시간 조명 체험(FEVER LAB) 및 브랜드 가치 탐색**
1. **메인 페이지 접근:** 첫 진입 시 자동으로 크로스페이드되는 히어로 이미지 슬라이더와 마키 밴드 애니메이션 및 마우스 커서를 따라다니는 콘페티 트레일 감상
2. **브랜드 가치 탐색:** 스크롤 다운을 통해 3단계 친환경 공정 인포그래픽 확인
3. **FEVER LAB(실습실) 이동:** 실시간 조명 커스텀 캔버스 및 컨트롤러 섹션 진입
4. **갓 패턴 커스텀:** 갓 패턴 칩을 클릭하여 패턴을 실시간으로 교체하며 젤리 팝핑 인터렉션 피드백 확인
5. **3단 터치 디밍 조절:** 밝기 스위치를 클릭하여 램프 밝기의 실시간 변화 확인
6. **조명 기둥 각도 조절:** 각도 조절 슬라이더를 마우스로 조작해 기둥을 실시간으로 구부려 보고, 방향에 맞춰 움직이는 스위치 체인의 물리 반응 체험
7. **OOTD 갤러리 감상:** 갤러리 영역 진입 시 스크롤 반응에 따라 전체 웹사이트 테마가 자동으로 다크 모드로 반전되는 사용자 경험 체득 및 룸투어 카드 찜하기 버튼 조작
8. **제품 구매 유도:** 페이지 하단의 에픽 CTA 상자 혹은 모바일 전용 하단 고정 바의 '입양하기' 버튼을 선택해 구매 프로세스로 연결

## 5. AI 활용 및 개발 워크플로우 (Vibe Coding)

* **초기 구조 설계:** 비대칭 인터록킹 그리드와 다중 백그라운드 이미지 슬라이더 등 난이도가 높은 UI의 CSS 구조와 React 상태 설계를 Antigravity AI와 협업하여 신속하게 프로토타이핑했습니다.
* **인터랙션 및 모션 튜닝:** 스크롤에 따른 다크모드 반전과 마우스 트레일 컨페티 입자들의 메모리 누수 방지 코드(animationend 시점에 remove() 처리)를 AI의 코칭을 받아 완성도 높게 튜닝했습니다.

## 6. 핵심 구현 기능
* **실시간 대화형 조명 시뮬레이터 (Fever Lab):** 사용자의 입력에 따라 패턴, 디밍, 각도가 부드러운 트랜지션과 바운싱 애니메이션으로 반응하는 대화형 인스턴스
* **스크롤 반응형 다크 테마 존:** Intersection Observer API를 활용하여 갤러리 섹션에 진입할 때 웹사이트 전체 테마를 동적으로 다크 모드로 반전시키는 시각적 임팩트 선사
* **동적 마우스 컨페티 트레일:** 데스크탑에서 마우스의 이동 속도와 위치에 반응하여 다채로운 파스텔 톤의 도형들이 튀어나와 부드럽게 낙하하는 인터랙션 구현
* **반응형 웹 디자인(RWD):** 데스크톱의 비대칭 그리드 레이아웃부터 모바일 환경을 고려한 반응형 처리 설계

## 8. 트러블 슈팅 (Troubleshooting)

### 이슈 1: 특정 영역 경계에서 화면 테마가 깜빡이는 현상
* **상황:** 스크롤이 다크 모드 영역의 경계선에 걸쳐 있을 때, 미세한 스크롤 움직임에도 밝은 테마와 어두운 테마가 빠르게 번갈아 켜지며 화면이 깜빡이는 현상이 발생했습니다.
* **해결:** 화면이 어두운 구역에 충분히 들어왔을 때만 테마가 바뀌도록 감지 기준을 완화하고 여유 공간을 확보하여, 마우스 휠 조작 시에도 깜빡임 없이 자연스럽게 테마가 전환되도록 수정했습니다.

## 9. 회고 및 인사이트

**[기술보다 중요한 배려, 친환경 가치와 사용자 경험]**
친환경이라는 브랜드 미션을 사용자에게 주입식 텍스트가 아닌 인터랙티브 시뮬레이션과 인포그래픽을 통해 '재미있게 체득'할 수 있도록 돕는 것이 훌륭한 프론트엔드 UX 설계임을 깨달았습니다.

**[AI와의 협업, Vibe Coding의 가능성]**
Vibe Coding의 핵심은 개발자의 기획/디자인 방향성에 AI의 강력한 연산력과 리팩토링 능력을 페어링하여 개발 생산성을 한 차원 끌어올리는 데 있습니다.
특히 물리 스위치의 움직임이나 디밍에 따른 수치 계산 등 세세한 스타일 수정을 AI를 활용하여 시행착오를 대폭 단축했습니다. 도구를 도구 이상으로 활용하여 아이디어를 현실화하는 최고의 워크플로우를 경험했습니다.
