# PRD (Product Requirement Document)

## 1. 프로젝트 개요
* **프로젝트명:** CHECKERBOARD FEVER 브랜드 랜딩 페이지
* **목적:** 친환경 비건 실리콘 및 체커보드 패브릭 장스탠드 조명의 시각적 매력과 친환경적 가치를 전달하고, 핵심 타깃의 소장 욕구를 자극하여 전환(입양하기)을 유도하는 힙한 브랜드 랜딩 페이지 구축
* **주 타깃 (Persona):** 김민지 (27세, 여성, 브랜드 마케터) - 가치 소비(미닝아웃)를 지향하면서도 감각적이고 위트 있는 데스크테리어 제품에 지갑을 여는 소비층
* **핵심 메세지:** "지구를 사랑하는 가장 키치한 방법, 내 방을 밝히는 체커보드 무드."
* **디자인 시스템 요약:**
    * **컬러:** 버터 크림 옐로우 (`#FFF2AC`, 배경), 포레스트 딥 그린 (`#0B4628`, 주조), 코랄 탠저린 (`#FF6B35`, 강조)
    * **폰트:** `Sofadi One` (강조용 영문), `Pretendard` (본문용 국/영문)
    * **모션 컨셉:** 젤리 팝핑 (Jelly Popping) 인터랙션 (탄성 있고 쫀득하게 튕기는 마이크로 모션)

---

## 2. 기술 스택 & 개발 환경
* **기본 프레임워크:** React (v18+) + Vite
* **스타일링:** Vanilla CSS (일반 `.css` 파일 활용, CSS Variables 적극 활용)
* **개발 가이드라인:**
    * Figma 디자인 시스템의 토큰(컬러, 폰트)을 글로벌 CSS 변수로 선언하여 일관성 유지
    * 컴포넌트 기반 설계를 통해 각 섹션 및 인터랙티브 요소(배지, 버튼, 탭)의 재사용성 확보

---

## 3. 글로벌 디자인 시스템 세팅 (CSS Variables)

`src/index.css` 또는 공통 스타일 파일에 아래 스펙을 기본 선언하고 작업을 시작한다.

```css
:root {
  /* Color Tokens */
  --bg-butter: #fff2ac;
  --primary-forest: #0b4628;
  --accent-coral: #ff6b35;
  --pure-white: #ffffff;
  
  /* Typography Tokens */
  --font-heading: 'Sofadi One', cursive;
  --font-body: 'Pretendard', -apple-system, BlinkMacSystemFont, system-ui, sans-serif;
  
  /* Interaction Tokens (Jelly Popping 치트키) */
  --ease-jelly: cubic-bezier(0.175, 0.885, 0.32, 1.275);
  --transition-jelly: transform 0.4s var(--ease-jelly), box-shadow 0.4s var(--ease-jelly);
}
```

## 4. 섹션별 기능 요구사항 및 세부 레이아웃

### 4.1 Section 1: Hero (첫화면)
* **레이아웃:**
    * 데스크톱: 비대칭 2단 그리드/플렉스 레이아웃 (좌측 타이포+CTA / 우측 조명 오브제 그래픽)
    * 모바일: 수직 1열 배치로 자동 전환
* **포함 요소:**
    * `Sofadi One` 기반 영문 타이틀: "CHECKERBOARD FEVER"
    * 서브 카피: "지구를 사랑하는 가장 키치한 방법, 내 방을 밝히는 체커보드 무드."
    * 강조색(`--accent-coral`) 바탕의 대형 CTA 버튼: `[내 책상에 입양하기]`
* **모션 및 인터랙션:**
    * **진입 시 (Page Load):** 우측 조명 이미지가 `scale(0)`에서 `scale(1)`로 쫀득하게 튕기며 등장. 좌측 카피는 아래에서 위로 스르륵 페이드인.
    * **호버 시:** CTA 버튼 호버 시 `scale(1.08, 0.95)` 형태로 젤리처럼 늘어나며 커짐. 클릭 시(`:active`) 꾹 눌리는 반동 피드백 구현.

### 4.2 Section 2: Brand Value (가치 소비 제품성)
* **레이아웃:** 지그재그 교차 배치 형태의 카드 레이아웃
* **포함 요소:**
    * 헤드라인: "지구를 생각하는 마음은 생각보다 말랑하니까."
    * 핵심 속성 카드 2개: 비건 실리콘 소개 카드, 친환경 인증 재생 자수 원사 패브릭 카드
    * 알약 모양 배지 컴포넌트: `Non-Toxic`, `Eco-Certified` 
* **모션 및 인터랙션:**
    * **스크롤 시:** 컴포넌트가 화면 영역에 진입할 때 알약 배지들이 상단에서 툭툭 떨어지듯 차례대로 팝핑되며 등장.
    * **호버 시:** 카드 영역에 마우스를 올리면 각진 블랙 그림자(`box-shadow`)가 더 두꺼워지며 카드가 위로 통통 튀어 오름.

### 4.3 Section 3: Product Detail (인터랙티브 디테일)
* **레이아웃:** 캔버스 영역(좌)과 조절형 탭 메뉴(우)로 구성된 인터랙티브 실험실 레이아웃
* **포함 요소:**
    * 헤드라인: "만지고, 구부리고, 바꾸고. 내 손끝에서 켜지는 텍스처 무드."
    * 실시간 반응형 조명 그래픽 컴포넌트
    * 사용자 조작용 UI 컴포넌트: 조명 기둥 각도 조절 바, 3단 디밍(밝기) 스위치, 갓 패턴 선택 칩 세트
* **모션 및 인터랙션 (React State 연동 핵심):**
    * **갓 패턴 체인지:** 패턴 칩을 클릭하면 React 상태가 변경되면서 조명 갓의 이미지가 부드럽게 회전하듯 팅- 튕기며 교체됨.
    * **터치 디밍 조절:** 3단 스위치 클릭 시 스위치 토글이 쫀득하게 굴러가며, 조명 그래픽 주변의 백그라운드 그라데이션 명도가 실시간으로 스위칭됨.

### 4.4 Section 4: OOTD Gallery (공간 매칭)
* **레이아웃:** CSS Grid를 활용한 Airbnb 스타일의 정교한 엇박자 반응형 갤러리 레이아웃
* **포함 요소:**
    * 헤드라인: "방 불이 꺼지면, 나만의 체커보드 파티가 시작됩니다."
    * 실제 타깃 유저 인테리어/데스크테리어 감성 연출 컷 (격자 배치)
    * 각 이미지 카드 우측 상단 코랄 오렌지색 하트(찜하기) 아이콘 컴포넌트
* **모션 및 인터랙션 (공간 반전 효과):**
    * **배경색 스위칭 (스크롤 트리거):** 해당 섹션 진입 시 전체 사이트 배경색이 버터색(`--bg-butter`)에서 포레스트 딥 그린(`--primary-forest`)으로 은은하게 반전되며, 갤러리 내 조명 이미지들이 밤에 켜진 듯 극적으로 강조됨.
    * **하트 클릭:** 찜하기 하트 버튼 클릭 시, 하트가 풍선처럼 통- 터지듯 커졌다 제자리로 돌아오는 마이크로 인터랙션 구현.

### 4.5 Section 5: Epic CTA (마무리 결제 유도)
* **레이아웃:** 볼드한 테두리로 감싸진 미니멀 디자인의 전면 중앙 정렬 레이아웃
* **포함 요소:**
    * 최종 문구: "내 방의 밀도를 바꾸는 가장 귀여운 오브제."
    * 초대형 젤리형 CTA 버튼: `[내 책상에 입양하기]`
* **모션 및 인터랙션:**
    * 버튼 내부 화살표 문구가 일정한 주기로 오른쪽으로 튕기며 유저의 마지막 행동을 강력히 유도.

### 4.6 [추가] Section 2.5: Brand Story (위트 있는 미닝아웃)
* **레이아웃:** 좌측 스토리 텍스트 / 우측 친환경 공정 프로세스 3단계 인포그래픽 (가로 배치)
* **포함 요소:**
    * 헤드라인: "예쁜 건 좋은데, 지구한테 미안하긴 싫어서."
    * 내용: 폐페트병이 수거되어 세련된 체커보드 자수 원사로 재탄생하는 3단계 미니멀 일러스트.
* **모션 및 인터랙션:**
    * 스크롤 진입 시 1단계부터 3단계까지 화살표가 젤리처럼 늘어나며 차례대로 활성화되는 스크롤 애니메이션.

### 4.7 [추가] Section 4.5: Unboxing & Spec (소장 가치 부여)
* **레이아웃:** 2열 그리드 (좌측: 재생지로 만든 키치한 패키지 및 굿즈 이미지 / 우측: 미니멀한 스펙 테이블)
* **포함 요소:**
    * 100% 생분해 사탕수수 친환경 패키지 스펙, 젤리 스티커팩 포함 구성 안내.
    * 제품 상세 스펙 (사이즈, 크래들 무게, 전선 소재 등 데스크테리어 유저 맞춤 정보).
* **모션 및 인터랙션:**
    * 패키지 이미지 호버 시 박스가 쫀득하게 들썩이며 내부 스티커 굿즈들이 부드럽게 페이드인되는 마이크로 인터랙션.

## 5. 모바일 대응 가이드 (RWD)
* **Breakpoint:** 단일 브레이크포인트 `768px` 기준 미디어 쿼리 적극 활용.
* **터치 UI 최적화:** 모바일 기기 특성상 마우스 호버(`:hover`)를 감지할 수 없으므로, 손가락으로 컴포넌트를 터치하거나 스와이프할 때 일시적으로 쫀득하게 찰랑거리는 CSS 액티브 상태 애니메이션으로 대체 적용.
* **하단 스티키 바:** 모바일 뷰 스크롤 시 화면 하단에 코랄 오렌지색의 `[내 책상에 입양하기]` 미니 젤리 버튼이 상시 플로팅 스티키 (`position: sticky; bottom: 0;`) 형태로 고정 노출되도록 구현하여 전환율 방어.
